🔐 Cybersecurity Threat Intelligence Dashboard (Power BI)
📌 Overview

This project is a Security Operations Center (SOC)-style dashboard built in Power BI to analyze cybersecurity network activity. It visualizes attack patterns, severity levels, and temporal trends to support threat monitoring and security analysis.

📊 Features
Analysis of cybersecurity attack types (e.g., malware, DDoS, phishing)
Severity level classification of security incidents
Time-series visualization of attack trends
Interactive filtering using slicers (attack type, severity, date range)
KPI tracking for total security events
🧰 Tools & Technologies
Power BI Desktop
Power Query (data cleaning & transformation)
DAX (data modeling and measures)
CSV-based cybersecurity log dataset
📈 Key Insights
Identified most frequent cybersecurity attack types
Analyzed distribution of severity levels across incidents
Detected trends and spikes in network attack activity over time
Enabled interactive exploration of threat data using filters
